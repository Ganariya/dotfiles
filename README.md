# How to Use

### cloneと実行権限


```bash
cd ~
git clone https://github.com/Ganariya/dotfiles.git
cd dotfiles
chmod +x install.sh
chmod +x install_brew.sh
chmod +x install_other.sh
```

### 設定ファイルのインストール

```bash
./install.sh
```

### HomeBrew関連のインストール

```bash
./install_brew.sh
```

### その他のインストール

```bash
./install_other.sh
```

### fishのインストール

```bash
./install_fish.sh
```

### 言語のインストール

```bash
./install_language.sh
```
